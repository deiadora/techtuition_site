## the framework

Techtuition is built on two interlocking structures:

### the multiscalar framework
The same principles apply at every scale simultaneously:
- **individual** ↔ **team** ↔ **organisation**
- **self** ↔ **family / chosen family / close friends** ↔ **community**
- **node** ↔ **network** ↔ **ecosystem**

It's fractal thinking with practical implications. The governance patterns, collaboration rituals, and operational structures that emerge from it cohere at every scale because they describe the actual architecture of coherent systems — not a methodology invented for the workplace.

### the quantum sequence (0–9)
Ten principles mapped to the body, the group, and the collective. Each principle has a geometric form, a quantum physics grounding, and a physiological centre in the body. The sequence is not linear — all ten are simultaneously present. What changes is which one is most legible at any given moment.

See *Quantum Currency* for the full architecture.
